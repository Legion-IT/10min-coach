# 10-Minute Coach · 10-минутный тренер 🏋️

**Bilingual (Русский / English)** — переключатель языка RU/EN в шапке; язык определяется
по браузеру и запоминается.

Веб-приложение (PWA) по программе **«30 минут в день с гантелями и лавочкой»**:
утро + день + вечер по 10 минут. Показывает, **что делать прямо сейчас** —
по дню недели и времени суток, ведёт по кругам с **таймером**, звуком и голосом,
и к каждому упражнению рисует **картинку**. Устанавливается на телефон как иконка
и **работает офлайн**.

> **🔗 Онлайн:** <https://devops.legion-it.services/coach/>
> **📦 Репозиторий:** <https://github.com/Legion-IT/10min-coach>
> Разместить (nginx на VM `legion-landings`): `/var/www/devops/coach/`.

## Возможности
- **Два языка (RU / EN)** — переключатель в шапке, автоопределение по браузеру, запоминается.
- **Фоновая музыка** — оригинальные зацикленные клезмер-треки на утро/день/вечер (сгенерированы ElevenLabs, инструментал), тихо играют под тренировку; кнопка 🎵 вкл/выкл, запоминается.
- **«Сейчас»** — сам определяет день недели (Пн–Вс) и блок (утро/день/вечер).
- **Плеер-таймер** — разминка → 2 круга по 4 упражнения → отдых, с кольцом обратного отсчёта.
- **Звук + голос** (рус.) — сигналы на 3-2-1, «смените сторону», объявление упражнений. Можно выключить 🔇.
- **Картинки** ко всем упражнениям (векторные, чёткие на любом экране).
- **Экран не гаснет** во время тренировки (Wake Lock).
- **Полностью офлайн** после первого открытия (Service Worker).
- **Лёгкие дни** (Ср/Сб) и **восстановление** (Вс) — с отдельными сценариями.
- Справка: как прогрессировать, техника, когда остановиться, забота о коленях.

## Запуск локально (на компьютере)
```powershell
cd C:\workspace\10min-coach
python -m http.server 8010
```
Открыть в браузере: <http://127.0.0.1:8010/>

> Service Worker и установка PWA работают только по `http://localhost` или по `https://`.
> Открытие файла напрямую (`file://`) — без офлайна и установки.

## Установка на Android (иконка на экране «Домой»)
1. Откройте на телефоне в **Chrome**: <https://devops.legion-it.services/coach/>
2. Меню (⋮) → **«Установить приложение»** (или само появится предложение).
3. На экране появится иконка «Тренер 10 мин» — открывается как обычное приложение,
   без адресной строки, работает офлайн.

**iPhone/Safari:** «Поделиться» → «На экран „Домой“».

### Где разместить (HTTPS)
- **Netlify Drop** — самый быстрый: перетащите папку (или `10min-coach.zip`) на
  <https://app.netlify.com/drop> → получите ссылку вида `https://…netlify.app`. Аккаунт не обязателен.
- **GitHub Pages / Cloudflare Pages / Vercel** — если есть аккаунт.
- **Свой сервер** (nginx и т.п.) — просто скопировать файлы в веб-каталог.

Приложение написано с **относительными путями**, поэтому работает и в корне сайта,
и в подпапке (например, `https://site.ru/coach/`).
